## First Steps

### Clone

#### HTTPS

```bash
git clone https://github.com/bastean/x.git && cd x
```

#### SSH

```bash
git clone git@github.com:bastean/x.git && cd x
```

### Initialize

#### Locally

1. System Requirements

   - [Go](https://go.dev/doc/install)
   - [Task](https://taskfile.dev/installation)

2. Run

   ```bash
   task init
   ```
